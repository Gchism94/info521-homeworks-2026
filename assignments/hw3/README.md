# Homework 3

This homework has two components.

1. **Programming**. The first component This repository contains a [Jupytext
   notebook](https://jupytext.readthedocs.io/en/latest/), similar to the one
   you saw in the previous homework.
2. **Written**. The second component is the written portion, in which
   you will answer a few questions and include a few plots that you will
   generate in `hw.py`. The LaTeX code to do this has already been provided
   for you in `hw.tex`, all you need to do is to
   1. actually generate the plots by implementing the correct code in `hw.py`,
   2. add a descriptive caption to the figures (wherever you see `ADD CAPTION HERE`
      in `hw.tex`), and
   3. uncomment the relevant figure environment lines in `hw.tex`.

### Submission

In order to complete your submission, you will need to commit and push the
following:

- The updated `hw.py` file that contains your code and answers to the
  questions within.
- The updated `hw.tex` file containing your answers.
- The compiled `hw.pdf` file containing the plots, captions, and answers to the
  questions.
