# README

## Instructions

Compile `hw.tex` to a PDF (see the `README.md` file for HW0 for how to do
this), and complete all the exercises in it.

See the `README.md` file for HW1 for instructions on how to install required
packages, working with Jupytext notebooks, etc.

The solutions to the exercises in `hw.pdf` need to be written up and submitted
by the beginning of class on the due date. The solutions can be hand-written,
with the exception of (i) the figure and caption required for Exercise 1, and
(ii) the answers to the workload and acknowledgments questions, which need to
be added to `hw.tex`.

# Grading

- 1 point: Homework is complete and correct (code passes all automated
  correctness tests, PR is not merged, existing docstrings have not been
  edited or deleted).
- 0.5 points: Homework is incomplete or has errors.
- 0 points: Homework was not submitted on time.

### Submission

In order to complete your submission, you will need to:
- commit and push the updated `hw.py` file.
- commit and push the updated `hw.tex` file.
- commit and push the generated `sample_based_approximation.pdf` file.
- commit and push the compiled `hw.pdf`. This file must contain the figure and
  caption for Exercise 1, as well as your workload estimation and
  acknowledgments.
- The steps for the analytical computation of the expectation in Exercise 1, as
  well as the solution for Exercise 2 may be written up by hand or
  typeset with LaTeX in `hw.tex` itself, whichever you prefer. If you are submitting a
  hand-written set of solutions, you must submit a hard copy to me at the
  beginning of class on the due date.
